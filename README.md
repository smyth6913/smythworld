# SmythWorld IT Solutions

A modern, highly professional, responsive single-page corporate website for an IT agency named "SmythWorld IT Solutions".

## Key Technologies
- **Framework**: TanStack Start
- **UI Library**: React 19
- **Routing**: TanStack Router v1
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Deployment**: Netlify

## How to Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server using Netlify CLI for full feature emulation:
   ```bash
   npx netlify dev
   ```
   Or using standard Vite dev server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` (or the port provided by `netlify dev`) to view the application.

## Features
- Deep corporate blue/charcoal design with vibrant accent highlights.
- Smooth scrolling navigation.
- Responsive, modern grid layout for core services.
- Engaging About Us and Contact sections with interactive forms.
